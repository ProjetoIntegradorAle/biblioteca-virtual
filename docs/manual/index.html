Bem-vindo à Biblioteca Virtual Alexandria

A Biblioteca Virtual Alexandria funciona como repositório digital acessível para materiais acadêmicos produzidos pelos estudantes do campus IFRN.

Imagem - página “Sobre a Biblioteca Virtual Alexandria”.

Público-alvo e Objetivo

  • Público: A biblioteca foi desenvolvida para estudantes do campus.
  • Objetivo: O objetivo da Biblioteca Virtual é permitir que os estudantes compartilharem seus materiais acadêmicos, ao mesmo tempo que garante que outros usuários tenham acesso, facilitando a consulta e a colaboração acadêmica.
